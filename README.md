## 📖 Introduction
안녕하세요! **AI 연구자**를 꿈꾸는 컴퓨터공학도입니다.

최근 **AI와 Deep Learning** 분야에 깊은 관심이 생겨 본격적인 학습을 시작했습니다. 단순히 이미 만들어진 라이브러리(API)를 호출하는 것에 그치지 않고, 수식으로만 접했던 딥러닝의 핵심 메커니즘을 **바닥부터** 직접 코드로 구현해보며 **"어떻게, 그리고 왜 작동하는가"**를 깊이 있게 체득하는 과정을 기록합니다.

이 저장소는 **밑바닥부터 시작하는 딥러닝 1**을 학습하며 작성한 소스 코드와 챕터별 핵심 요약을 포함하고 있습니다.

## 🛠 Tech Stack
- **Language**: Python 3.x
- **Libraries**: NumPy, Matplotlib
- **Tools**: VS Code, Git

## 📂 Curriculum & Progress
학습한 챕터는 체크박스(✅)로 표시합니다.

- [✅] **Ch01. 헬로 파이썬**: Python 문법 적응 및 NumPy 브로드캐스팅(Broadcasting)의 효율성 이해
- [✅] **Ch02. 퍼셉트론**: 단층 퍼셉트론의 한계(XOR 문제)와 다층 퍼셉트론의 필요성
- [ ] **Ch03. 신경망**: 활성화 함수(Sigmoid, ReLU)의 역할과 행렬 곱(Dot Product)을 통한 순전파 구현
- [ ] **Ch04. 신경망 학습**: 손실 함수(Cross Entropy Error)와 수치 미분을 통한 기울기 산출
- [ ] **Ch05. 오차역전파법**: 계산 그래프(Computational Graph)를 이용한 효율적인 역전파(Backpropagation) 구현
- [ ] **Ch06. 학습 관련 기술들**: 다양한 Optimizer(SGD, Adam 등)와 가중치 초기화, 배치 정규화의 중요성
- [ ] **Ch07. 합성곱 신경망(CNN)**: 이미지 처리를 위한 Convolution/Pooling 계층의 구조와 `im2col` 구현
- [ ] **Ch08. 딥러닝**: 심층 신경망의 역사와 VGG, GoogLeNet, ResNet 구조 이해
